# Code Templates

Link đến code templates cho copy-paste nhanh.

---

## Location

Code templates trong folder: `ios-template-docs/05-CODE-TEMPLATES/`

---

## Available Templates

| File | Purpose |
|------|---------|
| `01-REDUCER-TEMPLATE.swift` | TCA Reducer template |
| `02-VIEW-TEMPLATE.swift` | SwiftUI View template |
| `03-SERVICE-TEMPLATE.swift` | Service template |
| `04-REPOSITORY-TEMPLATE.swift` | Repository template |
| `05-API-CLIENT-TEMPLATE.swift` | API client template |
| `06-TEST-TEMPLATE.swift` | Unit test template |
| `07-PREVIEW-TEMPLATE.swift` | SwiftUI preview template |

---

## Usage

```bash
# Copy template
cp ios-template-docs/05-CODE-TEMPLATES/01-REDUCER-TEMPLATE.swift \
   Sources/Features/MyFeature/MyFeatureReducer.swift

# Find & replace "FeatureName" with your feature name
```
