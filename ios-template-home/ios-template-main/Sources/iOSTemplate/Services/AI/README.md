# AI Services

## Overview

This directory contains AI integration services for cloud-based AI providers.

## Services

### OpenAI Integration
- `OpenAIModels.swift` - Data models for OpenAI API
- `OpenAIService.swift` - OpenAI API client implementation

### Claude Integration
- `ClaudeModels.swift` - Data models for Anthropic Claude API
- `ClaudeService.swift` - Claude API client implementation

### Unified Interface
- `AIManager.swift` - Unified interface for all AI providers

## Usage

See `/docs/AI_INTEGRATION_GUIDE.md` for complete documentation.

## Environment Setup

```bash
export OPENAI_API_KEY=sk-...
export ANTHROPIC_API_KEY=sk-ant-...
```
