# ML Services

## Overview

This directory contains on-device machine learning services using Core ML and Vision frameworks.

## Services

### Core ML
- `CoreMLManager.swift` - Core ML model loading and prediction

### Vision
- `VisionService.swift` - Computer vision features (OCR, face detection, barcode scanning, etc.)

## Features

- Text recognition
- Object detection
- Face detection with landmarks
- Barcode/QR scanning
- Image classification
- Body/hand pose detection (iOS 14+)
